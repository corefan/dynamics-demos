# pdb-cloth
Cloth Simulation based on Position Dynamics solver.
WIP
Several demos showing:
* Basic Distance and Collision constraints
* Force fields (wind)
* Iterative solver
* Slow motion (with Stiffness adjustment)
* Normal computing
* Self-collisions
* Force based response to collisions
